No Man's Sky - Universal Font

As far as we have observed, there have been two fonts that carry the iconic NMS typographic style: the "NMSGeoSansLight" webfont in use on the official NMS site, and an alternative version datamined from the game files that seems to be based on the Futura Pro font family. This project aims to bring to creators the fixed versions of the fonts.

Getting in touch

Please use the Discussions forum to get in contact with us, discuss questions or offer help.

If you would like to report bugs, please open an issue or pull request on the specific repository. If you are not sure which repository is affected, feel free to create a post on the Discussions forum.

You can also send an email to support@nmsassistant.com, even though this is an email address for AssistantNMS support, all AssistantNMS Developers, Designers, etc... are part of the NMSCD Organisation and should be able to help you.
